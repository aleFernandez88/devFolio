## 📸 Preview

> Dark-themed single-page portfolio with smooth scroll animations, flip-box project cards, and a typing hero section.

---

## 🚀 Features

- **Typing animation** on the hero section via Typed.js
- **Scroll-triggered animations** with WOW.js & Waypoints.js
- **Animated skill bars** showing proficiency levels
- **Flip-card project showcase** with Isotope filtering
- **Responsive layout** built on Bootstrap 4 grid
- **Dark theme** with cyan accent (`#66fcf1`) and 12 gradient presets
- **Smooth scroll** navigation between sections
- **Pre-loader** and back-to-top button
